
Podemos criar variáveis para o CSS. Em uma programação responsiva, se um atributo é muito repitido no CSS
torna-se necessário atribuí-lo a uma variável.

:root {
    --cor-primaria: #000000;
    --cor-secundaria: #F6F6F6;
    --cor-terciaria: #22D4FD;
    --cor-hover: #272727;

    --fonte-primaria: 'Krona One', sans-serif;
    --fonte-secundaria: 'Montserrat', sans-serif;
}

Exemplo de aplicação de variável:
body{
    box-sizing: border-box;
    background-color: var(--cor-primaria);
    color: var(--cor-secundaria);
}
Aplique as variáveis no style.css

Outra medida responsiva para  adotar são as unidades medidas relativas (em, rem, vh) em vez de medidas absolutas (cm, mm, in, px).
Substitua o font-size: em px por rem.
