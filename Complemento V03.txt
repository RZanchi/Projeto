Mudunças para fazer aos poucos no CSS, para tentar entender o que é alterado a cada comando adicionado.
Focaremos nas classes .apresentacao__links, .apresentacao__links__subtitulo e .apresentacao__links__link

Na classe .apresentacao__links e .apresentacao__links__subtitulo, faremos as seguintes mudanças:

.apresentacao__links{
    display: flex;
    flex-direction: column;
    justify-content: space-between;
    align-items: center;
    gap: 32px;
}
.apresentacao__links__subtitulo{
    font-family: 'Krona One', sans-serif;
    font-weight: 400;
    font-size: 24px;
}

O flex-direction: column, muda o alinhamento dos botões de linha (padrão do flexbox) para coluna.

.apresentacao__links__link{
    /* background-color: #22D4FD; */
    border: 2px solid #22D4FD;
    width: 378px;
    text-align: center;
